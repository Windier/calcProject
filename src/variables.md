# Welcome to StackEdit!

$x,y,z \rightarrow$ **Cartesian**
$r,\theta,z \rightarrow$ **Cylindrical**
$u,v \rightarrow$ **Parametric**
$t \rightarrow$ **Time**

**Surfaces and Curves**
Explicit
* Curves
	* Cartesian ($x,y,z$)
		* $[y,z] = f(x) = x^2$
		* $[x,z] = f(y) = y^2$
		* $[x,y] = f(z) = z^2$
	* Polar
		* $r = f(\theta)$
* Surfaces
	* Cartesian
		* $z = f(x,y) = sin(x+y)$
		* $y = f(z,x) = sin(z+x)$
		* $x = f(z,y) = sin(z+y)$
	* Cylindrical
		* $z = f(r,\theta)$
		
Parametric
* Curves
	* $[f(u),g(u)]$
* Surfaces
	* $[f(u,v),g(u,v),h(u,v)]$
	
**Vector fields**
* $\vec{v} = [a,b,c]$;
* $\vec{v}(x,y) = [P(x,y),Q(x,y)]$
* $\vec{v}(x,y,z) = [P(x,y,z),Q(x,y,z),R(x,y,z)]$
* $\vec{v}(x,y,z) = [p(x,y,z),q(x,y,z),r(x,y,z)] \rightarrow [P(x,y,z),Q(x,y,z),R(x,y,z)]$
